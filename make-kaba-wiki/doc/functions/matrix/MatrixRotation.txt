// Description
Erstellt eine Matrix, die einen Vektor um den Ursprung dreht.\nDer Vektor wird um die Achsen gedreht in der Reihenfolge (Z,X,Y), wobei die Drehachsen nicht mitgedreht werden.
// Parameter m_out
out
Die Rotations-Matrix.
// Parameter ang
in
Die Drehwinkel um die jeweiligen Achsen sind die Komponenten des Vektors (im Bogenma&s).
// Synopsis
s:vector
