// Description
Erstellt ein Dialog-Fenster, dessen Größe variabel ist.
// Parameter title
in
Der Text in der Titelleiste.
// Parameter width
in
Die Fensterbreite.
// Parameter height
in
Die Fensterhöhe.
// Parameter root
in
Das übergeordnete Eltern-Fenster.
// Parameter allow_root
in
Das Elternfenster soll weiterhin benutzbar sein, obwohl der Dialog ausgeführt wird.
