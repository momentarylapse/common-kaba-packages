// Description
Ändert die aktuellen Bildschirm/Auflösungs-Einstellungen.
// Parameter api
in
Die Grafik-API s(<c>"OpenGL"</c> oder <c>"None"</c>).
// Parameter xres
in
Die Bildschirmauflösungs-Breite in Pixeln.
// Parameter yres
in
Die Bildschirmauflösungs-Höhe in Pixeln.
// Parameter fullscreen
in
Vollbildmodus. <c>true<c> startet den Vollbildmodus, <c>false<c> lässt die Engine im Fenster laufen.
#
