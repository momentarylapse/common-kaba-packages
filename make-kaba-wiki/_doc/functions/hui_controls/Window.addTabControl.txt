// Description
Fügt dem Fenster ein Tab-Feld hinzu.
// Parameter title
in
Die Aufschrift (mit "\\" werden die einzelnen Tab-Elemente getrennt).
// Parameter x
in
Die X-Position.
// Parameter y
in
Die Y-Position.
// Parameter width
in
Die Breite.
// Parameter height
in
Die Höhe.
// Parameter id
in
Die ID, sie diehnt zur Idendifizierung der Schaltfläche und als Nachricht, das dem Fenster von der Schaltfläche geschickt wird (wenn das aktuelle Tab-Feld geändert wird)).
