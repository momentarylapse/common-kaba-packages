// Description
Fügt dem Fenster eine Auswahlliste hinzu.
// Parameter title
in
Die Aufschrift (durch "\\" werden die einzelnen Auswahlelemente getrennt).
// Parameter x
in
Die X-Position.
// Parameter y
in
Die Y-Position.
// Parameter width
in
Die Breite.
// Parameter height
in
Die Höhe.
// Parameter id
in
Die ID, sie diehnt zur Idendifizierung der Schaltfläche und als Nachricht, das dem Fenster von der Schaltfläche geschickt wird (wenn die Auswahl geändert wird).
