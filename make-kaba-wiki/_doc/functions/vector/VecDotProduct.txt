// Description
Berechnet das Skalarprodukt zweier Vektoren ("inneres Produkt", "Punktprodukt") (<math> \langle v1 , v2 \rangle </math>), es ergibt sich aus <math> \operatorname{Länge}(v1) \cdot \operatorname{Länge}(v2) * \cos ( \operatorname{WinkelZwischen}(v1,v2) ) </math>, ist also <math>=0</math> für orthogonale (senkrechte) Vektoren.\nEs gilt (<math>> \langle v1 , v2 \rangle = \langle v2 , v1 \rangle </math>).
// Parameter v1
in
Erster Vektor
// Parameter v2
in
Zweiter Vektor.
// Return
Das Skalarprodukt, es wird berechnet durch <math> v1.x \cdot v2.x + v1.y \cdot v2.y + v1.z \cdot v2.z </math>.
#
