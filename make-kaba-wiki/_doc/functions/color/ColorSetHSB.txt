// Description
Berechnet eine Farbe aus Farbwert, Sättigung und Helligkeit.\nAlle Parameter sind aus dem Intervall [0..1].
// Parameter a
in
Alpha. ("inverse Transparenz")
// Parameter h
in
Farbwert. Gibt den Punkt im Farbspektrum ab.
// Parameter s
in
Sättigung. 0 entspricht einem Grauton, 1 der maximalen Farbintensität.
// Parameter b
in
Helligkeit. 0 entspricht schwarz, 1 der hellstmöglichen Farbe.
// Return
Der errechnete Farbwert.
// Synopsis
s:color,color,ColorInterpolate,ColorSetHSB
#
