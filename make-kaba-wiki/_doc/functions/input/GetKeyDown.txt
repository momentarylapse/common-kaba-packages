// Description
Testet, ob eine Taste gerade gedrückt wurde (also in diesem Frame gedrückt ist, es aber im letzten noch nicht war).
// Parameter id
in
Index der zu prüfenden Taste. (z.B. <c>KEY_ESCAPE</c> für die [Escape]-Taste)
// Return
<c>true</c>, falls gerade gedrückt worden, ansonsten <c>false</c>.
#
