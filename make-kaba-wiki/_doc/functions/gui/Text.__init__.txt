// Description
Erschafft eine Schrift-Anzeige auf dem Bildschirm. (relative Koordinaten!)
// Parameter pos
in
Position der linken oberen Ecke des Textes. Wird <v>centric</v> gesetzt, so wird diese Position zur Mitte der oberen Kante.
// Parameter size
in
Höhe der Schrift als Bruchteil der Fenster-/Bildschirmhöhe.
// Parameter c
in
Die Farbe der Schrift,
// Parameter str
in
Der anzuzeigende Text.
// Synopsis
s:Text
#
