// Description
Erschafft ein Anzeige-Element für 3d-Modelle auf dem Bildschirm.
// Parameter m
in
Das anzuzeigende 3d-Modell.
// Parameter mat
in
Die Transformations-Matrix des Modelles, die die Modell-Vertices in die [0,1]x[0,1]-Koordinaten des Bildschirms projeziert.\n<c>m_id</c> diehnt als Identität (Einheits-Matrix).
// Parameter z
in
Die Tiefe der Anzeige, sie muss in [0,1) liegen und wird f&ur die Darstellungs-Reihenfolge der Anzeigen verwendet.

s:Picture3d
